# Support

## Open Source Project

This is an open source project, developed and maintained by the community. We appreciate your interest and support!

## How to Get Support

If you need help, have a question, or want to report a bug, please use the GitHub Issues tracker.

### Reporting Issues

1.  **Search First**: Before opening a new issue, please search existing issues to see if your problem has already been reported or solved.
2.  **Open a New Issue**: If you can't find a solution, feel free to [open a new issue](https://github.com/vicajilau/quizdy/issues).
3.  **Provide Details**: When reporting a bug, please include as much detail as possible:
    *   Steps to reproduce the issue.
    *   Expected behavior.
    *   Actual behavior.
    *   Screenshots or error messages (if applicable).
    *   OS version and App version.

By using GitHub issues, you allow the entire community to benefit from the discussion and help solve the problem faster.

Thank you for contributing to the project!
